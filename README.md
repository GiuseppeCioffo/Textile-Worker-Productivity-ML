# Textile Worker Productivity – Machine Learning
This project explores a real-world industrial dataset from the garment manufacturing sector
to predict worker productivity using machine learning techniques.

## Project goal
The goal is to build and compare different ML models to understand which factors
most influence productivity and how well it can be predicted.

## Data
The dataset comes from a public industrial dataset related to garment manufacturing: https://archive.ics.uci.edu/dataset/597/productivity+prediction+of+garment+employees

## Approach
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model comparison
- Model evaluation and interpretation

## Technologies
Python, pandas, numpy, scikit-learn, matplotlib
